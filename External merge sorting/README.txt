Дано: 
2 файла с не пересекающихся случайными Int64 значениями, по одному значению на каждой строке, размер файлов не ограничен и может превышать размер доступной оперативной памяти 

Требуется: 
Получить третий  файл, который содержит значения из двух файлов выше и в котором  значения отсортированы по возрастанию, очевидно, что результирующий файл так же может превышать размер доступной оперативной памяти.

Реализация:
Дано: файл программы task.py, два исходных файла input_001.txt и input_002.txt
После запуска программы первым делом вызывается функция разделения файла на файлы поменьше part_file для обоих исходных файлов, в результате чего появляются файлы temporary-first.{m}.txt (деление первого файла) и temporary-second.{n}.txt (деление второго файла). Получившиеся файлы не превышают 5 строк, но при желании это можно изменить. Данные внутри каждого файла отсортированы.
Далее сначала для первого файла, потом для второго вызывается функция внешней сортировки combine_sorted_files. Прежде всего первые два файла temporary-{index} сортируются в {index}-output.1, после чего начинается сортировка остальных temporary-{index} файлов с использованием {index}-output.1.txt и {index}-output.2.txt. Каждый из последних двух файлов поочередно выступает то в роли исходного, то в роли выходного файла до тех пор, пока информация из всех temporary-{index} файлов не будет отсортирована.
После этого функция сортировки применяется к файлам {index}-output, в которые в последней итерации цикла выступали выходными файлами. В результате отсортированные данные записываются в файл final-output.txt, выводится сообщение о завершении.
